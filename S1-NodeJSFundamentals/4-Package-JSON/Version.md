Versions: Semantic Versioning
Modifer: ^, ~, *

^1.12.2
(modifier) Major.minor.patch

Patch: Bug Fixes

Minor: Don't mess up with other functionality

Major: Changes the functionality of the feature


Additive Changes: 
Patch: Bug fixes
Minor: Adding some feature which doesn't have any impact on other code

Destructive Changes: 
Major version bump


~: Auto upgrade patch version
^: Auto upgrades minor verion
*: Auto upgrades major version
